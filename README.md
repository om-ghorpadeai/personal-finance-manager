# 💼 Personal Finance Manager

A simple Python-based application to manage personal finances.

---

## 🚀 Features
- Add income 💰
- Add expenses 💸
- View financial records 📊
- Calculate current balance

---

## 🛠️ Technologies Used
- Python

---

## ▶️ How to Run
1. Download the project
2. Open terminal in project folder
3. Run:
   python main.py
